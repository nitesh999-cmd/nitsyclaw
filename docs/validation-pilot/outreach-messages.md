# NitsyClaw Pilot Outreach Messages

## Primary warm DM

```text
Quick one — you're always buried in bills, receipts and little admin reminders, right?

I built a personal admin assistant for myself and I'm testing whether it helps other people too.

I'm taking 2–3 people for a 2-week pilot:
- send me bills, receipts and things-to-remember however is easiest
- I keep the reminders organised
- I send you a clean weekly admin digest

AUD 39 for the two weeks, full refund if it does not save you time.

Want in? I can show you a real example first.
```

## Shorter WhatsApp version

```text
Quick test — I built a life-admin assistant for bills, receipts and reminders.

I'm taking 2–3 people for a 2-week pilot. You send messy admin, I return reminders + a weekly digest.

AUD 39, refunded if it does not save time. Want to see an example?
```

## If they ask what they can send

```text
Anything normal-life admin:

- bills with due dates
- receipts or expenses
- calls you need to remember
- renewal/cancellation reminders
- complaint drafts
- things you keep forgetting

No bank login, no mailbox access, and I will not send/pay/call on your behalf. I only organise, remind, summarise and draft.
```

## If they say "can't ChatGPT do this?"

```text
ChatGPT can help if you remember to open it, explain context, and manage the follow-up yourself.

This pilot is different: you send the messy admin as it happens, and I turn it into reminders, expense notes, bill summaries and a weekly digest.

The value is not just the AI answer — it is getting the admin out of your head and followed up.
```

## If they ask about privacy

```text
For the pilot, keep it simple:

- no bank login
- no email login
- no passwords or codes
- no IDs unless you choose to redact them first
- I only use what you send for the admin task

You can also black out anything sensitive before sending it.
```

## Follow-up after no reply, 2 days later

```text
No pressure — I'm only looking for 2–3 people to test whether this is actually useful.

If bills/receipts/reminders are not painful for you, all good.

If they are, I can run the 2-week pilot and refund if it does not save time.
```

## Close message if they decline

```text
All good. That helps too — I'm trying to work out if this is a real paid problem or just a cool tool.

If admin starts annoying you later, send me one bill/receipt/reminder and I'll show you what the digest would look like.
```
